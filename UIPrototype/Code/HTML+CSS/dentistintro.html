职称： 主任医师

科室： 口腔正畸科

专长： 牙体牙髓常见病疑难病诊断治疗，显微根管治疗，牙齿美白美容修复。

简介： 唐志辉，男，北京大学口腔医学院教授、主任医师、博士研究生导师，
　　　1993年、1999年毕业于北京大学口腔医学院，获口腔医学学士、博士学位，2001年至今任北京大学口腔医院第二门诊部主任，2007年任北京大学-香港大学牙周专业临床硕士(国际课程)项目管理委员会委员，2013年在美国密歇根大学牙医学院做访问学者。

研究方向： 主要集中在生物活性植骨材料、3D打印种植牙的研发和种植体周围病诊治方面。
　　　作为第一作者或通讯作者在国际、国内核心期刊杂志上发表了30余篇口腔种植学、干细胞方面的论文(SCI20篇)，参编教材2部，指导和协助指导北京大学口腔医学院硕士、博士、博士后30余名。获科技部、北京市科委、国际牙种植学会等国内、国际科研基金10余项，国家发明专利6项。
